HUA E-Studies Repository

#
Run web server via Docker locally
----------------------------------
** config file
```bash
cp .env.test .env
cp ./frontend/.test.env ./frontend/.env
cp ./admin_console/.test.env ./admin_console/.env
```

** docker start
```bash
docker-compose build
```

** docker running
```bash
docker-compose up
```

** docker end - remove
```bash
docker-compose down
```
